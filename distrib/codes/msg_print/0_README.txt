Exemplo de Chamada de Procedimento Remoto Usando Sun RPC

Este exemplo é uma adaptação do código dado na Seção "Remote Message Printing
Example" do site https://web.cs.wpi.edu/~rek/DCS/D04/SunRPC.html .

Este pacote contém os arquivos msg.x, rprintmsg.c, msg_proc.c e printmsg.c .
O conteúdo dos arquivos msg.x e rprintmsg.c está dado no site mencionado acima
(na seção citada).
O conteúdo do arquivo msg_proc.c foi modificado.
O arquivo printmsg.c foi adicionado.

Os passos para compilar e executar o código do exemplo estão dados no mesmo
site (no fim da seção citada).
Caso esteja usando o Ubuntu, você pode precisar instalar o pacote rpcbind .

Outra referência sobre chamada de procedimento remoto usando Sun RPC:
https://www.cs.rutgers.edu/~pxk/rutgers/notes/content/ra-sunrpc.pdf

Caso tenha alguma dúvida ou encontre algum erro neste exemplo, entre em
contato!
